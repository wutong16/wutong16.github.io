IDArb: Intrinsic Decomposition for Arbitrary Number of Input Views and Illuminations

December 2024
PDF Cite
Overall framework

Abstract

IDArb is a diffusion-based method for intrinsic decomposition from an arbitrary number of images under varying illuminations. It uses cross-view, cross-domain attention and illumination-augmented view-adaptive training to estimate multi-view consistent normals and material properties.
